# Data_Science_Games

Proyecto de análisis de datos sobre ventas de videojuegos.

## Descripción

Este repositorio contiene un notebook de Jupyter y un conjunto de datos que permiten explorar y analizar las ventas de videojuegos a nivel global. El objetivo principal es aplicar técnicas de ciencia de datos (por ejemplo regresión, visualización, clustering) para extraer conocimientos del mercado de videojuegos.

## Contenido del repositorio

| Archivo | Descripción |
|--------|------------|
| `vgsales.csv` | Conjunto de datos con ventas de videojuegos (global, por plataforma, género, región, etc.). |
| `gamesales (3).ipynb` | Notebook de Jupyter con análisis exploratorio, limpieza, visualización y modelos de predicción. |
| `desktop.ini` | Archivo del sistema (puede ignorarse). |

## Objetivos del análisis

- Realizar un análisis exploratorio de datos (EDA) del dataset de ventas de videojuegos.
- Visualizar las tendencias de venta por plataforma, región y género.
- Preparar los datos para modelos de predicción.
- Aplicar al menos un modelo de regresión para estimar ventas globales o en una región determinada.
- (Opcional) Aplicar clustering o segmentación para descubrir grupos de videojuegos similares por ventas.

## Tecnologías y herramientas

- Python (versión ≥ 3.8)
- Jupyter Notebook
- Bibliotecas: `pandas`, `numpy`, `matplotlib`, `seaborn`, `scikit-learn`
- Git / GitHub para gestión de versiones
